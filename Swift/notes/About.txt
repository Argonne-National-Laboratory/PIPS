= PIPS/Swift Development Manual

== Configuration

You currently need:

* SWIG
+
Install with:
+
----
apt-get install swig
----
+
* Swift/T
+
Cf. http://www.mcs.anl.gov/~wozniak/stc.html#_installation
+
* The swig-data app
+
----
svn co https://svn.mcs.anl.gov/repos/exm/apps/swig-data
cd swig-data
./build.sh
----
+

////
////

Run +./build.sh+
